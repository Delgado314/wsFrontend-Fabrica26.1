# Mr. Poopybutthole Archive

Catálogo de personagens de **Rick and Morty**. Busque por nome, navegue por páginas e abra a ficha de cada personagem.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- React · TypeScript
- Tailwind CSS
- [The Rick and Morty API](https://rickandmortyapi.com)

## Como rodar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

Outros comandos:

| Comando       | Descrição              |
| ------------- | ---------------------- |
| `npm run build` | Build de produção  |
| `npm run start` | Servidor pós-build |
| `npm run lint`  | ESLint               |

## Créditos

Dados e imagens vêm da API pública citada acima. Personagens e marcas pertencem aos respectivos proprietários.
